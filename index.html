A free Pexels API key lets this tool fetch fresh live photos instead of the built-in curated set. Get one in about two minutes at pexels.com/api — sign up, go to your profile, and copy the API key. It stays saved in your browser only; it is never sent anywhere except directly to Pexels.

This tool surfaces photos matched to common Jewish prayers — selected for beauty, thematic resonance, and clear space for text overlay. With a free Pexels API key (above), it fetches fresh live results. Without one, it uses a built-in curated set. Each prayer section also includes a direct Pexels search link.
